# 🧠 AI Memory Engine (Using Endee Vector DB)

## 📌 Project Overview
This project is an AI-powered memory system that stores and retrieves user thoughts using vector similarity search.

## 🚀 Features
- Store user memories
- Retrieve similar memories
- Semantic search

## 🛠️ Tech Stack
- Python
- Streamlit
- Endee

## ⚙️ How to Run
pip install streamlit  
streamlit run app.py  

## 📂 Use of Endee
Endee is used as vector database for semantic retrieval.
